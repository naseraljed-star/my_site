# Developer Guide

## 1. Project Overview
This project is a personal website for Naser Aljed, showcasing his profile as a Cybersecurity Student. It includes sections for an about me description, contact information, and links to external resources.

## 2. Language Used
The website is built using HTML and CSS.

## 3. Website Purpose
The purpose of the website is to present Naser Aljed's background in cybersecurity, provide contact information, and link to his GitHub profile and Google.

## 4. User Flow

```mermaid
flowchart TD
    A[User lands on homepage] --> B[View profile content]
    B --> C[Read About Me section]
    C --> D[View contact information]

    E -->|My GitHub| MyGitHub[My GitHub]
```

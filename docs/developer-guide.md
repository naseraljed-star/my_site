# Developer Guide for Naser Aljed's Personal Website

## 1) Project Overview
This project is a personal website for Naser Aljed, a cybersecurity student. It serves as an online portfolio and a platform to showcase interests in cybersecurity, particularly in secure coding and CI/CD pipelines. The website allows visitors to learn more about Naser and provides links to external resources.

## 2) Language Used
- **HTML** for structure
- **CSS** for styling

## 3) Website Purpose
The purpose of this website is to present Naser Aljed's professional identity, interests, and skills related to cybersecurity. It offers a simple and appealing user interface for visitors to contact him and explore his online presence through links to external sites like Google and his GitHub.
